Fedoro - Digital Newspaper

Creating your own digital newspaper sounds like a huge task. There are plenty of frameworks to choose from and many of them actually are not specifically designed for this porpuse.

For someone with skills in PHP, SQL as well as JavaScript modifying a framework like WordPress shouldn't be a huge problem. But you still have to do many things on your own.
Implementing an easy interface for your Authors, managing registered users, having different article categories. And even if you manage to implement every feature you need, you will still have a big overhead because of WordPress

About

While trying to figure out which framework our school's newspaper could use to digitalize our articles, I quickly realized that none of the available Frameworks was specifically tailored for this purpose.
Having a good understanding of how SQL, PHP and the Boostrap framework works I decided to code a solution on my own.
Fedoro's main goal is to give users the opportunity to create newspaper websites that are easy to use and flexible for multiple users.
Because it was my last year at this school, the setup and maintenance had to be as straight-forward as possible

My contributions

  • I created to project from scratch using Bootstrap and PHP
  • Designing the database structure
  • Coding a flexible PHP interface
  • Designing the webpage
  • Connecting the Boostrap website to the SQL Database via PHP

Details

  • Using Bootstrap as a rendering Framework
  • Implemented using HTML, CSS, JavaScript, SQL and PHP
  • Supports many different designs and languages (can be extended by the community)
